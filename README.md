# agentic-workflow-snow-tickets
This repo holds the high-level plan for building a workflow to solve ServiceNow tickets with an agentic AI workflow. The infrastructure underlying this workflow will be deployed and managed by Terraform Enterprise. The Agentic AI workflows will be secured by HashiCorp Vault Enterprise. The ticket resolution workflows will be driven by ServiceNow.
